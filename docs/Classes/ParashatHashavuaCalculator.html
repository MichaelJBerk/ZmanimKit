ParashatHashavuaCalculator

public class ParashatHashavuaCalculator

The KCParashatHashavuaCalculator class calculates which parasha is read on the shabbat following a supplied date. It can also return the listing for an entire year.

In Israel, due to differences in holiday observance, the schedule may vary from the diaspora in some years. KCParashatHashavuaCalculator accounts for these as well, and can return the appropriate value for both Israel and the diaspora.
